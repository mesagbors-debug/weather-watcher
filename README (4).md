

# 🌦️Weather Watcher
A fast, modern weather app that gives you real time local weather and a 7 day forecast right in your browser.
Powered by a free weather API, with a clean UI, responsive layout, and instant location search.

 ## Features
Real time weather data (temperature, wind, conditions, highs/lows)

7 day forecast with icons and daily summaries

Search any city worldwide

“Use My Location” button for instant local weather

Sunrise & sunset times

Clean, modern UI with weather based icons

Fully responsive layout for mobile and desktop

## How It Works
Enter a city name or use your current location

The app fetches live weather data from the API

Current conditions + a 7 day forecast are displayed

Icons and colors update based on weather type

Everything updates instantly without reloading the page
